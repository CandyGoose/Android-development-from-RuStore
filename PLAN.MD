# Декомпозиция MVP магазина приложений

## Эпик 1. Список приложений

План: реализация каталога на экране, переход в карточку.

Итого: около 60 ч (7,5 дн.).

| № | Задача | ч |
|---|--------|---|
| 1.1 | Согласовать поля ответа сервера для списка (идентификатор, имя, иконка, версия, краткое описание) | 4 |
| 1.2 | Модель предметной области для строки списка | 4 |
| 1.3 | Модели ответа сервера, преобразование в модель приложения, сетевой клиент и заглушка для отладки без сервера | 8 |
| 1.4 | Слой доступа к данным каталога (репозиторий) | 6 |
| 1.5 | Операция: выдать список приложений; пустой список и сбой сети | 4 |
| 1.6 | Зарегистрировать зависимости: репозиторий, операции, клиент | 4 |
| 1.7 | Экран списка: список, ячейка, состояния загрузки и ошибки | 8 |
| 1.8 | Модель экрана списка: загрузка, повтор, по нажатию - переход с идентификатором пакета | 6 |
| 1.9 | Переход список - карточка, аргументы сохраняются при повороте | 4 |
| 1.10 | Модульные тесты операции или модели экрана | 8 |
| 1.11 | Потянуть список для обновления, мелкие правки интерфейса | 4 |

## Эпик 3. Установка и удаление

Планы: установка и удаление приложений, статус "установлено" в списке и карточке.

Итого: около 60 ч (7,5 дн.).

| № | Задача | ч |
|---|--------|---|
| 3.1 | Выбрать способ установки по заданию курса, зафиксировать решение | 8 |
| 3.2 | Код запуска установки, отмена пользователем | 8 |
| 3.3 | По системному менеджеру пакетов узнать, установлен ли пакет, обновить экраны | 8 |
| 3.4 | Удаление: стандартный сценарий Android, результат где требуется | 6 |
| 3.5 | Манифест и разрешения, проверка на нужных версиях Android | 4 |
| 3.6 | Сообщения пользователю при ошибках | 6 |
| 3.7 | Автотесты на устройстве или список шагов для ручной проверки | 8 |
| 3.8 | Описать в README: шаги сборки, требования к устройству и окружению, пошаговый сценарий ручной проверки установки | 4 |
| 3.9 | Проверить список и карточку: после установки и после удаления отображаются актуальный статус и кнопки, при возврате на экран и смене конфигурации данные не устаревают | 8 |
